# Embedding `circular.wongfu.net`

> Single URL → context-aware behavior.
> First introduced: **v3.0.49 (2026-05-27)**

The same URL `https://circular.wongfu.net/` automatically adapts based on
how it's consumed:

| Consumer | What appears |
|---|---|
| Direct visit (browser address bar) | Full splash + dashboard |
| Pasted in Slack / WhatsApp / Discord / Twitter | OG preview card (auto-generated by each platform from the OG meta) |
| Embedded via `<iframe>` | Chrome-less dashboard view (no top nav, no footer) with a small ↗ "完整版" corner link |

No URL params, no special embed endpoint, no config — the page detects its
own context using `window.self !== window.top`.

---

## 1. Minimal iframe embed

Paste this anywhere — that's it:

```html
<iframe
  src="https://circular.wongfu.net/"
  style="width:100%;border:0;border-radius:12px;min-height:700px;"
  loading="lazy"
  title="EDB 通告智能分析系統"
  allow="clipboard-read; clipboard-write"></iframe>
```

The dashboard auto-detects it's in an iframe and:
- Hides the top brand banner, tab navigation, and footer
- Adds a small floating "↗ 完整版" link in the bottom-right corner
  that opens the full dashboard in a new tab
- Posts its content height to the parent window (so the parent can
  auto-resize the iframe — see section 2)

## 2. Auto-resize iframe (optional)

The dashboard sends a `postMessage` to the parent window whenever its
content height changes. Listen for it on the host page:

```html
<iframe
  id="edbCircularFrame"
  src="https://circular.wongfu.net/"
  style="width:100%;border:0;border-radius:12px;min-height:700px;"
  loading="lazy"
  title="EDB 通告智能分析系統"></iframe>

<script>
  window.addEventListener('message', (e) => {
    // Only accept messages from circular.wongfu.net
    if (e.origin !== 'https://circular.wongfu.net') return;
    if (e.data?.type !== 'edb-circular-height') return;
    const frame = document.getElementById('edbCircularFrame');
    if (frame && e.data.height) {
      frame.style.height = e.data.height + 'px';
    }
  });
</script>
```

The message payload shape:

```js
{ type: 'edb-circular-height', height: <number-in-px> }
```

Sent on: `load`, `ResizeObserver` body resize, and twice after load
(500ms + 2000ms) to catch late-rendering content.

## 3. Sharing as a link (no iframe)

The OG metadata (added in **v3.0.48**) is configured for:
- `og:title` — EDB 通告智能分析系統
- `og:description` — AI 即時解讀教育局通告，自動提取摘要、識別截止日期與角色職責
- `og:image` — `https://circular.wongfu.net/assets/og-image.png` (1729×910)
- `og:url`, `og:site_name`, `og:locale=zh_HK`
- `twitter:card=summary_large_image`, `twitter:image`

Just pasting `https://circular.wongfu.net/` into Slack / WhatsApp /
Discord / Teams / Twitter will render a rich preview card automatically.

## 4. Manual project card (portal.wongfu.net showcase tile)

If you're building a project showcase grid and want a static tile that
opens the dashboard in a new tab on click:

```html
<a href="https://circular.wongfu.net/"
   target="_blank" rel="noopener"
   class="project-card">
  <img src="https://circular.wongfu.net/assets/og-image.png"
       alt="EDB 通告智能分析系統"
       loading="lazy">
  <div class="card-body">
    <h3>EDB 通告智能分析系統</h3>
    <p>AI 即時解讀教育局通告，自動提取摘要、識別截止日期與角色職責。</p>
```

Style it freely — the dashboard makes no assumption about how the parent
card is presented.

## Detection logic (for reference)

Implemented in both `index.html` and `edb-dashboard.html`:

```js
// Top of script, before splash animation / landing redirect
try {
  if (window.self !== window.top) {
    // We are inside an iframe → forward / adapt
    sessionStorage.setItem('edb_visited_landing', '1');
    window.location.replace('edb-dashboard.html');  // index.html only
    // (dashboard adds .is-embedded body class + postMessage instead)
  }
} catch (e) {
  // Cross-origin access throws → we ARE in an iframe
}
```

Cross-origin parent access throws a `SecurityError` — that exception
itself is a positive iframe signal, so the catch branch also forwards.

## Compatibility

- Modern evergreen browsers: full support
- Older browsers without `ResizeObserver`: dashboard still renders correctly
  in embed mode; only the auto-resize message stream degrades to "load-time
  notification only" (still useful)
- `backdrop-filter` (used by the "↗ 完整版" corner): Safari 14+, Chrome 76+,
  Firefox 103+. Falls back to a solid dark navy pill on older browsers.
